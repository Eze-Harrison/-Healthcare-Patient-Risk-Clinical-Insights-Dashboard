# Project Overview

In this project, I analyzed a synthetic healthcare dataset containing 10,000 patient records to uncover clinical patterns, patient risk levels, disease distribution, and hospital utilization trends.The goal of this analysis was to simulate a real-world healthcare analytics project by cleaning messy healthcare data, performing exploratory data analysis, and creating an interactive dashboard to generate meaningful insights. I generated the data using ChatGpt and Gemini AI by giving it prompts that depicts real world events.

# Analysis Questions

The project answers key healthcare analytics questions such as:

Which diagnoses are most common among patients?

 Which conditions have the highest average glucose levels?
 
 Which insurance types have the highest admission days?
 
 Which age group contributes most to disease occurrence?
 
 Are there relationships between glucose level and hospital admission?
 
 Which patients fall into high-risk categories?

## Key Insights

Some major findings from the analysis:

- COPD was the most common diagnosis among patients.
  
- Diabetes recorded the highest average glucose level.
  
- Patients aged 61+ showed the highest disease contribution.
  
- Male patients represented the majority of records.
  
- A significant percentage of patients were classified as high-risk.
  
- Certain insurance categories recorded higher hospital utilization.

## Recommendations

Based on the findings:

- Healthcare providers should prioritize monitoring high-risk patients.
  
- Preventive health programs should target older age groups.
  
- Patients with elevated glucose levels should receive early intervention.
  
- Hospital resources can be optimized based on disease patterns.
